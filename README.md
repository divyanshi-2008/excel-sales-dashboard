# 📊 Sales Dashboard in Microsoft Excel

## 📌 Project Overview

This is my first Excel dashboard project built using **Microsoft Excel 365**.

The goal of this project was to analyze a real-world sales dataset and create an interactive dashboard that helps understand sales performance across different months, countries, and products.

This project helped me practice data cleaning, Pivot Tables, Pivot Charts, calculated columns, and dashboard design.

---

## 📂 Dataset

- **Source:** Kaggle
- **Records:** 531,151+ sales transactions
- **Fields Included:**
  - Transaction Number
  - Date
  - Product Number
  - Product Name
  - Price
  - Quantity
  - Customer Number
  - Country
